# 数字

## calc

数字计算工具类

```typescript
import { calc } from '@fang-kang/js-utils';

const a = 2;

const b = 1;

calc(`${a}+${b}`); // 3
calc(`${a}-${b}`); // 1
calc(`${a}*${b}`); // 2
calc(`${a}/${b}`); // 2

// 格式化

calc('1').format(); // 1.00
```

## uppercaseNum

返回大写数字

```typescript
import { uppercaseNum } from '@fang-kang/js-utils';

uppercaseNum(1); // '一'
```
