isBrowser ​

检查是否为浏览器环境

可用于确定当前运行时环境是否为浏览器。这有助于避免在服务器（节点）上运行前端模块时出错。

typescript
import { isBrowser } from '@fang-kang/js-utils';

isBrowser(); // true (browser)
isBrowser(); // false (Node)
